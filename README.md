## 快速上手

下载 zip 包，将 **.zip** 改成 **.jar**，然后 **add lib** 即可使用。  

```java
// 针对单个 class
ForwardEngine.runClass("com.neko.entity.SystemUser");

// 或者 

// 扫描 package
ForwardEngine.runPackage("com.neko.entity");
```
